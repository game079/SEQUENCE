# 順序 (SEQUENCE)

バラバラに並んだ数字を、1から順番にクリックしていくシンプルなパズルゲームです。
短時間で遊べて、集中力・注意力を鍛える脳トレゲームとしても楽しめます。

## ゲームの特徴

3つの難易度: 簡単（EASY）、普通（NORMAL）、難しい（HARD）
直感的な操作: 数字をクリックするだけのシンプルなルール
レスポンシブ対応: PC・スマホどちらでもプレイ可能
タイムアタック要素: どれだけ早く正確にクリアできるか挑戦

## 遊び方

1. 難易度を選択します。
2. 画面に表示された数字を「1」から順番にクリックします。
3. 全ての数字を消せばクリアです。

## ルール

* 間違った数字をクリックしてもゲームは続行されます
* 正確さとスピードの両方を意識してプレイしましょう

## 開発環境

* HTML5 / CSS3 / JavaScript
